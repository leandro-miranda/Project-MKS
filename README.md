## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/leandro-miranda/Project-MKS-Frontend-challenge
   ```
2. Install NPM packages
   ```sh
   npm install
   or
   yarn
   ```

### Run

```sh
npm run dev
or
yarn dev
```
